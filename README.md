# LiveCodeEditor

LiveCodeEditor is a real-time collaborative code editor where multiple users can join a room and write code together.

## Features
- Multiple users can join a room
- Real-time code synchronization
- Syntax highlighting
- Room ID sharing
- Simple UI

## Tech Stack
- React.js
- Node.js
- Express.js
- Socket.io

## Run Locally
npm install  
npm start  
npm run server:dev
